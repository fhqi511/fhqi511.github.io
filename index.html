赵祺个人博客搭建HEXO

赵祺个人博客搭建HEXO

<提前准备:GIT>

下载nodejs

https://nodejs.org

第一步：准备

（基于mac10.15.5系统）

终端：sudo su //管理员登陆

node -v 查看node版本

npm install -g cnpm –registry=https://registry.npm.taobao.org

查看版本 cnpm -v

第二步：下载

cnpm install -g hexo-cli //下载镜像

hexo -v 查看版本

第三步：创建

新建文件夹 blog

mk dir 新建文件夹

pwd 查看当前的位置

ls -l查看所有文件

rm -rf * 删除所有文件

进入qiblog 此处一定要自己mkdir一个文件夹要不然会不为空，出现可避免的错误

之后 sudo hexo init 初始化一个hexo的镜像文件 sudo 管理员模式

第四步：安装

创建完成之后：

hexo s 启动 本地地址为 localhost:4000

hexo n “此处写文章名称 就是titile”

cd ..退一层级

Cd ../..退两层文件夹

hexo clean 清理内存，方便我们重新部署

安装插件：

cnpm install –save hexo-deployer-git

编辑config文件

hexo d 部署到github上

下载主题：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

新建仓库：fhqi511.github.io

名字严格控制 fhqi511就是你的登陆名
